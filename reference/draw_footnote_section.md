# Draw footnote section at a given bottom edge

Draw footnote section at a given bottom edge

## Usage

``` r
draw_footnote_section(grob, y_bottom_npc)
```

## Arguments

- grob:

  The footnote textGrob, or NULL.

- y_bottom_npc:

  Bottom edge of the footnote in npc units (outer_vp). With just =
  c(just, "bottom"), the bottom of the text aligns here.
