# Draw header section (left, center, right grobs) at a given top edge

Draw header section (left, center, right grobs) at a given top edge

## Usage

``` r
draw_header_section(grobs, y_top_npc)
```

## Arguments

- grobs:

  Named list from build_section_grobs(): header_left, header_center,
  header_right.

- y_top_npc:

  Top edge of the header row in npc units (outer_vp).
