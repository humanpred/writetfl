# Draw a rule grob at the vertical midpoint of a padding gap

Draw a rule grob at the vertical midpoint of a padding gap

## Usage

``` r
draw_rule(rule, y_mid_npc)
```

## Arguments

- rule:

  FALSE or a linesGrob (from normalize_rule()).

- y_mid_npc:

  Y midpoint of the padding gap in npc (outer_vp).
