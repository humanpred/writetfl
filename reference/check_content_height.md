# Check content height against minimum and collect error if too short

Check content height against minimum and collect error if too short

## Usage

``` r
check_content_height(content_h_in, min_content_height, errors)
```

## Arguments

- content_h_in:

  Computed content height in inches.

- min_content_height:

  A unit object.

- errors:

  Character vector to append to.

## Value

Updated errors character vector.
