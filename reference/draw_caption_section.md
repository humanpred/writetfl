# Draw caption section (single full-width grob) at a given top edge

Draw caption section (single full-width grob) at a given top edge

## Usage

``` r
draw_caption_section(grob, y_top_npc)
```

## Arguments

- grob:

  The caption textGrob, or NULL.

- y_top_npc:

  Top edge of the caption in npc units (outer_vp).
