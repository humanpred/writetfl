# Check figure height against minimum and collect error if too short

Check figure height against minimum and collect error if too short

## Usage

``` r
check_figure_height(fig_h_in, min_figheight, errors)
```

## Arguments

- fig_h_in:

  Computed figure height in inches.

- min_figheight:

  A unit object.

- errors:

  Character vector to append to.

## Value

Updated errors character vector.
