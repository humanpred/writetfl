# Resolve gp for a specific section/element from the gp hierarchy

Resolve gp for a specific section/element from the gp hierarchy

## Usage

``` r
resolve_gp(gp, section, element)
```

## Arguments

- gp:

  A gpar() or named list.

- section:

  One of "header", "caption", "figure", "footnote", "footer".

- element:

  One of "header_left", "header_center", "header_right", "caption",
  "footnote", "footer_left", "footer_center", "footer_right".

## Value

A gpar() object.
