# Draw footer section (left, center, right grobs) at a given bottom edge

Draw footer section (left, center, right grobs) at a given bottom edge

## Usage

``` r
draw_footer_section(grobs, y_bottom_npc)
```

## Arguments

- grobs:

  Named list from build_section_grobs(): footer_left, footer_center,
  footer_right.

- y_bottom_npc:

  Bottom edge of the footer row in npc units (outer_vp).
