# Measure widths of footer left/center/right grobs

Measure widths of footer left/center/right grobs

## Usage

``` r
measure_footer_widths(grobs)
```

## Arguments

- grobs:

  Named list from build_section_grobs() — expects footer_left,
  footer_center, footer_right.

## Value

Named list: left, center, right (numeric inches or NULL).
