# Measure a gt grob's height in a scratch device

Measure a gt grob's height in a scratch device

## Usage

``` r
.gt_grob_height(grob, pg_width, pg_height)
```

## Arguments

- grob:

  A gtable grob from
  [`gt::as_gtable()`](https://gt.rstudio.com/reference/as_gtable.html).

- pg_width, pg_height:

  Page dimensions for the scratch device.

## Value

Numeric scalar: grob height in inches.
